Задание 1

Сколько типов данных существует в JS?
Ответ: 6: числа, строки, логический, null, undefined и объекты. +1 - символы.

Как вывести информацию в консоль?
Ответ: с помощью console.log.

Какая функция операторов || и &&?
Ответ: для изменения или комбинации логических значений.



Задание 2

Какое будет выведено значение: let x = 5; alert( x++ ); ?
Ответ: 5 (а потом уже инкремент)

Чему равно такое выражение: [ ] + false - null + true ?
Ответ: NaN (сначала пойдет строка, но потом минус, используемый для цифр, все подпортит)

Что выведет этот код: let y = 1; let x = y = 2; alert(x); ?
Ответ: 2 (сначала y = 2, а потом x = y)

Чему равна сумма [ ] + 1 + 2?
Ответ: "12" (строка, состоящая из пустой строки, единицы и двойки)

Что выведет этот код: alert( "1"[0] )?
Ответ: 1 (первый элемент строки)

Чему равно 2 && 1 && null && 0 && undefined ?
Ответ: null (1. 2 && 1 вернет последнее - 1, а дальше null испортит смысл и будет превращать все в null)

Есть ли разница между выражениями? !!( a && b ) и (a && b)?
Ответ: два восклицательных знака всегда переведут значения в логический тип

Что выведет этот код: alert( null || 2 && 3 || 4 ); ?
Ответ: 3 (1. 2 && 3 вернет последнее - 3, 2. null || 3 вернет 3, так как null - false, значит посмотрит второе значение - 3, 3) 3 || 4 вернет 3, так как сразу 3 воспримется как true) 

a = [1, 2, 3]; b = [1, 2, 3]; Правда ли что a == b ?
Ответ: нет, 2 разных объекта

Что выведет этот код: alert( +"Infinity" ); ?
Ответ: Infinity (число плюс бесконечность из-за унарного плюса, приводящего строки к числу)

Верно ли сравнение: "ёжик" > "яблоко"?
Ответ: да, произойдет сравнение кодов символов

Чему равно 0 || "" || 2 || undefined || true || falsе ?
Ответ: 2 (1. 0 || "" вернет false, 2. false || 2 вернет 2, а при дальнейших или будет возвращаться 2, воспринимаемая, как true)
